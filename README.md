# Stroke Rehabilitation Exercise Monitoring

> AI-powered vision system for monitoring stroke rehabilitation exercises using human pose estimation and machine learning.

---

## Overview

Stroke rehabilitation requires repetitive therapeutic exercises under the supervision of physiotherapists. Continuous manual observation is expensive, time-consuming, and difficult to scale.

This project develops a computer vision system capable of automatically monitoring rehabilitation exercises using pose estimation and machine learning techniques.

The long-term objective of this project is to contribute toward AI-powered Rehabilitation Robotics.

---

## Motivation

Every year millions of stroke survivors require physical rehabilitation.

Traditional rehabilitation relies heavily on physiotherapists.

Recent advances in Computer Vision allow rehabilitation exercises to be evaluated automatically using only a camera.

This project investigates the feasibility of applying pose estimation and machine learning to rehabilitation monitoring.
